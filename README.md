后端部署:

编译Go程序: go build -o logviewer

运行: ./logviewer

前端部署:

安装依赖: npm install

开发模式: npm run serve

生产构建: npm run build (输出到dist目录)

配置:

修改config/config.yaml添加你的服务器信息

功能特点
多服务器支持:

通过配置文件管理多个服务器和日志文件

支持SSH密码和密钥认证

实时日志查看:

使用tail -f命令实时跟踪日志文件

通过WebSocket实时推送日志内容到前端

命令执行:

支持在远程服务器上执行任意命令

保存命令历史记录

用户界面:

响应式设计，适应不同屏幕尺寸

可同时查看多个日志文件

扩展建议
